Double-click any word to view its definition in a pop-up bubble Fork of the "Dictionary Anywhere" addon initally made to make it usable for android but now adds a couple of more features/improvements Short Demo Video: https://github.com/igorlogius/dbl-click-dict/assets/67047467/ad61d99a-22e4-43c1-8694-4ae14a2b3813 Features/Usage - Will show the definition of any word in a small pop-up bubble, when double-clicking it. - adds caching to show already viewed definitions quicker and without making anothe request - uses async/await syntax for clearer code structure - lets you add a meta key to the double click action - allows export of looked at definitons - choose between English, French, German and Spanish as the base language
